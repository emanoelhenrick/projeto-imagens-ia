O que é o Gerador de Imagens com IA?

Este projeto, desenvolvido para a disciplina de Introdução à Computação do curso de Análise e Desenvolvimento de Sistemas da Cesar School, tem como objetivo aplicar os conhecimentos iniciais de HTML e CSS por meio da criação de uma página web estática. A proposta busca introduzir os fundamentos do desenvolvimento front-end.

Gerador de Imagens com IA. Projeto para a disciplina de Introdução a Computação.
